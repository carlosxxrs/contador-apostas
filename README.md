# Contador de Apostas

Aplicativo Flask para registrar apostas com login de usuário.

## Como rodar localmente

1. Entre na pasta do projeto.
2. Instale as dependências:
   `pip install -r requirements.txt`
3. Execute:
   `python app.py`
4. Acesse:
   `http://127.0.0.1:5000`

## Deploy sugerido

- Railway
- Fly.io
- PythonAnywhere
- Heroku

Use o `Procfile` e garanta que `app.py` esteja na raiz do projeto.
